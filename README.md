# 🎬 YouTube Clone (Mini Project)

🚀 This is a **YouTube Clone** built using **HTML & CSS**.
It replicates the basic UI of YouTube including layout, navbar, and video grid.

---

## 📌 Features

✨ Modern YouTube UI Design
✨ Navigation Bar with Icons
✨ Video Thumbnail Grid Layout
✨ Sidebar (Home, Trending, Subscriptions)
✨ Clean & Responsive Design

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## 📂 Project Structure

```
youtube-clone/
 ├── index.html
 ├── style.css
 ├── img/
 ├── History.html
 ├── Library.html
 ├── Subscriptions.html
 └── Trending.html
```
